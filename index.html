PROJECT SHINOBI

A new take on Naruto

Project Shinobi is an Open World RPG Fan Game based on the popular anime/manga series Naruto. Version 3 of the demo is the latest and please bare in mind that the the game is still in early development, and won't be too advanced. The demos and final game will be free and some features of the game include:
  • Running and Flying as Naruto
  • Exploring the Hidden Leaf Village
  • Fighting Itachi and Sand Shinobi
  • Using Photo Mode
  • Creating Your Own Skill Tree
  • Summoning Gamakichi
  • Transforming into Kurama
